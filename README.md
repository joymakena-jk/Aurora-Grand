 Aurora Grand

Aurora Grand is a luxury-inspired front-end hotel website that showcases elegance, comfort, and premium hospitality through a modern digital experience. The project was built to strengthen my understanding of HTML, CSS, and JavaScript by creating an immersive, responsive, and visually engaging website from scratch.

Features

* Cinematic hero section with typewriter animation
* Responsive navigation bar with smooth scrolling
* Luxury Rooms showcase
* Azure Experience section
* Interactive image gallery with fullscreen lightbox
* Previous and Next gallery navigation
* Guest testimonial slider with automatic transitions
* Booking modal
* Scroll reveal animations
* Cursor glow effect
* Floating gold particle animation
* Spa & Wellness section
* Signature Dining section
* Modern hover effects and premium-inspired design

Built With

* HTML5
* CSS3
* JavaScript (Vanilla JavaScript)

Project Goals

The goal of this project was to:

* Practice semantic HTML structure
* Improve CSS layout and animation skills
* Strengthen JavaScript DOM manipulation and event handling
* Build a fully responsive website without using front-end frameworks such as Bootstrap
* Create a luxury-inspired user interface with interactive features and smooth user experience
